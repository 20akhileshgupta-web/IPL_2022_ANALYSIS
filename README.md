# IPL_2022_ANALYSIS

# 🏏 IPL 2022 Data Analysis Project (EDA)

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on the **IPL 2022 dataset** using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

The goal of this project is to analyze team performance, match trends, toss decisions, scoring patterns, and key insights from the IPL 2022 season.

---

## 🎯 Objectives
- Understand IPL 2022 match trends
- Analyze toss winning patterns
- Study first innings vs second innings scores
- Identify top-performing teams
- Visualize match-winning factors
- Generate meaningful business insights using EDA

---

## 📂 Dataset Information
The dataset contains IPL 2022 match details such as:

- Match ID
- Team 1
- Team 2
- Toss winner
- Toss decision
- Winner
- Venue
- First innings score
- Second innings score
- Won by (Runs/Wickets)
- Player of the Match

### 2. Univariate Analysis

- Match result distribution

### 3. Bivariate Analysis

- Bar Plot

- Box Plot
- Count Plot
- Heatmap

---

## 🔍 Key Insights

✅ Teams winning toss often preferred **fielding**

✅ Certain venues favored chasing teams

✅ First innings scores around **160–190** were common

✅ Toss had noticeable impact on match outcome

✅ Some teams consistently performed better in high-pressure matches
